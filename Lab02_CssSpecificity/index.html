CSS Specificity

CSS Specificity เหมือนเป็นค่าคะแนนบางอย่างที่สามารถช่วยให้บราวเซอร์ตัดสินใจได้ว่าจะ ใช้ css rules ใดในการ apply element ซึ่งวิธีการตัดสินใจก็คือจะเช็คค่าคะแนนจาก selector ของ rules นั้นๆ ถ้าหาก rules ใดมีค่าคะแนนมากที่สุด จะถูกนำไป apply element ซึ่งก็จะทำให้หน้าเว็บแสดงผล css style นั้นออกมา

โดยปกติแล้ว browserจะตัดสินใจเลือก rules ที่มาจาก external css ลำดับล่าสุดก่อน ถ้าเป็น rules ที่มาจาก external css เดียวกัน แล้วมีกรณีที่ rules มี selector เหมือนกัน แสดงว่ามีค่าคะแนนเท่ากัน จะเลือก selector ที่เป็นตัวที่ซ้ำกันล่าสุด เช่น

h1 {background-color: yellow;}
h1 {background-color: red;}

browser จะเลือก h1 {background-color: red;} ไป apply

Specificity Hierarchy

  • Inline styles เช่น E< h1 style="color: #d45252;">
    → มีค่าคะแนน 1000
  • ID เช่น #navbar
    → มีค่าคะแนน 100
  • Classes, attributes and pseudo-classes เช่น .class , [attributes] , :hover
    → มีค่าคะแนน 10
  • Elements and pseudo-elements เช่น ชื่อ element ต่างๆ (div, h1, p) และ (:before, :after)
    → มีค่าคะแนน 1

How to calculate

จำแนก slector แต่ละตัวใน rules ว่าแต่ละตัวมีค่าคะแนนเท่าไหร่ แล้วนำค่าคะแนนของทุกตัวมารวมกัน ก็จะได้ผลลัพธ์เป็นค่าของคะแนนรวม

*จริงๆแล้วค่าคะแนนดังกล่าวไม่ได้มีอยู่จริง เป็นเพียงการสมมุติเพื่อให้เกิดความเข้าใจได้ง่ายๆเท่านั้น

ตัวอย่างการคำนวณ

  • div#a {background-color: green;} → 1 + 100 = 101
  • #a {background-color: yellow;} → 100 = 100
  • div[id=a] {background-color: blue;} → 1 + 10 = 11

    • ผลลัพธ์ คือ div#a {background-color: green;} จะถูก apply เพราะมีค่าคะแนนมากที่สุด
