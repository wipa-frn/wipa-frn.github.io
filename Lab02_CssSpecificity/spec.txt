CSS Specificity
เหมือนเป็นค่าคะแนนบางอย่างที่สามารถช่วยให้บราวเซอร์ตัดสินใจได้ว่าจะ ใช้ css rules ใด ในการ apply element ซึ่งวิธีการตัดสินใจก็คือจะเช็คค่าคะแนนจาก selector ของ rules นั้นๆ 
ถ้าหาก rules ใดมีค่าคะแนนมากที่สุด จะถูกนำไป apply element ซึ่งก็จะทำให้หน้าเว็บแสดงผล css style นั้นออกมา

โดยปกติแล้ว browserจะตัดสินใจเลือก rules ที่มาจาก external css ลำดับล่าสุดก่อน 
 ถ้าเป็น rules ที่มาจาก external css เดียวกัน แล้วมีกรณีที่ rules มี selector เหมือนกัน แสดงว่ามีค่าคะแนนเท่ากัน จะเลือก selector ที่เป็นตัวที่ซ้ำกันล่าสุด เช่น 
h1 {background-color: yellow;}
h1 {background-color: red;} 
browser จะเลือก  h1 {background-color: red;} ไป apply


Specificity Hierarchy 
เรียงตามที่มีค่าคะแนนมากที่สุด
Inline styles เช่น  E<h1 style="color: #ffffff;"> มีค่าคะแนน 1000
ID เช่น  #navbar
Classes, attributes and pseudo-classes เช่น .class , [attributes] , :hover
Elements and pseudo-elements เช่น ชื่อ element ต่างๆ (div, h1, p) และ :before, :after
*ค่าคะแนนเป็นการสมมุติขึ้นมาเท่านั้น

มีค่าคะแนน 1000
มีค่าคะแนน 100
มีค่าคะแนน 10
มีค่าคะแนน 1

How to calculate

ตัวอย่าง
div#a {background-color: green;} --> 1+100 = 101
#a {background-color: yellow;} --> 100 = 100
div[id=a] {background-color: blue;} 1 + 10 = 11

ผลลัพธ์ คือ div#a {background-color: green;} จะถูก apply
